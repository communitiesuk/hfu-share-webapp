You will not have access to any data until your request is approved by MHCLG.

To request access to data

  1. You need to know which areas in the UK you need access to data for.
  2. Select ‘Start’.
  3. Answer the questions to find the most suitable access for you.
  4. Give a reason why you need access to the data. Give as much detail as possible.
  5. Your request is reviewed by admin in MHCLG.
  6. Check your home page for updates about access approval.

If your request is rejected you will be given a reason why. If you think you should have been given access you can submit another request with a more detailed reason why you need access.

If you make a mistake in your application you will need to submit a new request.
